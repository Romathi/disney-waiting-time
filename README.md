# Disneyland Paris Wait Times Dashboard

This project is a Streamlit dashboard that displays the wait times for attractions at Disneyland Paris. The dashboard is
built using Python and Streamlit, and it uses data from the Disneyland Paris API.

## Installation

To install the project, you need to have Python 3.10 or higher installed on your machine. You can then install the
project using pip:

```bash
pip install -r requirements.txt
```

## Usage

To run the project, you can use the following command:

```bash
streamlit run app.py

python main.py
```

The Streamlit dashboard will then be available at http://localhost:8501.

The `main.py` is used to fetch the data from the Disneyland Paris API and save it to a CSV file. The `app.py` is used
to display the data in a Streamlit dashboard.
